# Election Quest 🗳️

An immersive, 3D educational web application designed to teach the election process in both the **Indian National System** and the **Education System** (Student Council/Class Representative).

![Hero Screenshot Placeholder](https://via.placeholder.com/1200x600?text=Election+Quest+3D+Demo)

## ✨ Features

- **3D Visualizations**: A floating, interactive 3D Ballot Box powered by **Three.js**.
- **Voting Animation**: A custom GSAP animation where users "fold" their paper ballot and drop it into the box to start their journey.
- **Deep Educational Context**: "Chunky" content covering everything from Voter ID registration (Form 6) to the Model Code of Conduct.
- **Interactive Expandable Steps**: Accordion-style pointers to keep the learning experience clean and engaging.
- **Politi-Bot**: An integrated chatbot to answer democracy-related questions.
- **Smooth Navigation**: GSAP-powered smooth scrolling and micro-animations for a premium feel.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Glassmorphism)
- **3D Engine**: [Three.js](https://threejs.org/)
- **Animations**: [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)
- **Fonts**: Google Fonts (Outfit & Inter)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/election-quest.git
   ```
2. **Open the project**:
   Simply open `index.html` in any modern web browser.
   
*Note: Some 3D features and JS modules may require a local server (e.g., Live Server extension in VS Code).*

## 📖 What You'll Learn

### 🇮🇳 Indian National System
- Voter Registration & Eligibility
- Nomination Scrutiny (Form 26)
- Model Code of Conduct & Campaigning
- EVM/VVPAT Voting Process
- Counting & Result Declaration

### 🎓 Education System
- Student Nomination Criteria
- Drafting a Manifesto
- Public Speaking & Speeches
- Secret Ballot Voting
- Investiture & Responsibility
